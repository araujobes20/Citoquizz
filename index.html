📋 Como configurar — passo a passo
💡 As questões ficam direto na planilha do Google Sheets. Para adicionar, editar ou remover uma questão basta editar a linha correspondente — sem mexer no app, sem fazer download de nada.
① Crie uma planilha no Google Sheets com duas abas:
Questões ← onde você escreve as questões Resultados ← onde os resultados chegam automaticamente
② Na aba "Questões", coloque esses cabeçalhos na linha 1:
Enunciado | Categoria | Midia_Tipo | Midia_URL | Alt_A | Alt_B | Alt_C | Alt_D | Alt_E | Correta
• Midia_Tipo: escreva image, video, ou deixe em branco
• Midia_URL: cole o link da imagem/vídeo (ou deixe em branco)
• Correta: escreva apenas a letra: A, B, C, D ou E
③ Na aba "Resultados", coloque estes cabeçalhos na linha 1:
Timestamp | Nome | Matrícula | Acertos | Total | Aproveitamento | Score | Violações
④ Abra Extensões → Apps Script, apague tudo e cole este código:
function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('Questões'); var rows = sheet.getDataRange().getValues(); var out = []; for (var i = 1; i < rows.length; i++) { var r = rows[i]; if (!r[0]) continue; out.push({ text: r[0], category: r[1] || 'Geral', media: { type: r[2] || 'none', url: r[3] || '' }, options: [r[4], r[5], r[6], r[7], r[8]], correct: ['A','B','C','D','E'].indexOf(String(r[9]).trim().toUpperCase()) }); } return ContentService .createTextOutput(JSON.stringify(out)) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('Resultados'); var d = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toLocaleString('pt-BR'), d.name, d.matricula, d.correct, d.total, d.pct + '%', d.score, d.violations || 0 ]); return ContentService .createTextOutput('OK') .setMimeType(ContentService.MimeType.TEXT); }
⑤ Publique como Web App:
Clique em Implantar → Nova implantação
Tipo: App da Web  |  Executar como: Eu  |  Quem acessa: Qualquer pessoa
→ Copie a URL e cole no campo de configuração acima. Clique em Salvar e depois em Testar conexão.
⑥ Pronto! Para adicionar questões:
Abra a planilha → aba Questões → adicione uma linha por questão.
Na próxima vez que um aluno abrir o app, as questões novas já aparecem automaticamente.
